# 🚲 Bike Sharing Dashboard

Dashboard interaktif untuk menganalisis pola penggunaan bike sharing di Washington D.C. selama tahun 2011–2012 menggunakan Streamlit.

---

## ⚙️ Instalasi

1. Clone repository:

```
git clone https://github.com/lveichana/bike-sharing-dashboard.git
cd bike-sharing-dashboard
```

2. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Menjalankan Aplikasi

```
streamlit run dashboard/dashboard.py
```

Aplikasi akan berjalan di browser:
http://localhost:8501

---

## 🌐 Deploy

Dashboard dapat diakses secara online melalui Streamlit Cloud:
https://bike-sharing-dashboard-jihan.streamlit.app/

---

## 📌 Dataset

Dataset yang digunakan adalah Bike Sharing Dataset (2011–2012) dari Kaggle:
https://www.kaggle.com/datasets/lakshmi25npathi/bike-sharing-dataset
